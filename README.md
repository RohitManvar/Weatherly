# 🌦️ Real-Time Weather Web Application

A dynamic web app built with Java (JSP & Servlets) that provides real-time weather updates using external API integration. Users can input a city name to fetch and view live weather information including temperature, humidity, wind speed, and weather conditions.

## 🔍 Features

- 🔎 City-based weather search
- 🌡️ Live temperature, humidity, and wind speed display
- ☁️ Weather condition icons
- 💬 Error handling for empty or invalid city names
- 📱 Responsive UI for mobile and desktop
- 🌓 Light/Dark mode toggle (optional)

## 🛠️ Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript, JSP
- **Backend**: Java Servlet
- **Weather API**: [OpenWeatherMap](https://openweathermap.org/api)
- **Server**: Apache Tomcat (or any Java servlet container)
- **IDE**: NetBeans (or Eclipse/IntelliJ)
